## 📌 Description
Arduino project that sends an **SOS signal (... --- ...)** using a LED (and buzzer).  
It demonstrates basic Morse code and digital output control.

---

## 🧰 Materials
- Arduino Uno  
- LED + 220Ω resistor  
- Buzzer  
- LCD (I2C)  
- Push button  
- Breadboard + wires  

---

## ⚙️ Usage
1. Connect LED → pin 10  
2. Upload the code  
3. Press the button  
➡️ The SOS signal is sent  

---

## 🚀 Features
- Morse code (SOS)  
- LED + buzzer output  
- Clean code with `dot()` and `dash()`
